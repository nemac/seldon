Seldon

An OpenLayers Web-GIS.

Seldon is a GIS viewer built upon the OpenLayers framework. Rendered with HTML5 & JavaScript it is compatible with modern web browsers and mobile devices.

Layers

Seldon can display multiple layers of data color-coded and in a legend for your analysis.

Masking
